# Report_Analysis-of-ED-tech-
Mini Power BI project

 Project Overview

This project is a Power BI dashboard created to analyze data and answer multiple business questions using interactive visualizations. The dashboard helps users understand trends, KPIs, and insights through different report pages.

---

##  Objectives

- Analyze the given dataset.
- Build interactive dashboards.
- Answer business-related questions.
- Improve decision-making using data visualization.
